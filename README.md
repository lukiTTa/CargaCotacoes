# CargaCotacoes
Aplicação em .NET desenvolvida para a disciplina de Sistemas Distribuídos, UNIFEI 2022.

# Descrição
Um web crawler periódico de dados de ações e de tesouro direto, com publicação no banco de dados MySQL.

# Tecnologias
O crawling foi realizado através do Selenium WebDriver no Chrome e as rotinas definidas através do FluentScheduler.
